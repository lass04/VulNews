     VULNEWS Fullstack Project

   * a Cybersecurity informative fullstack project with MEAN Stack
   presents Latest Articles,CVEs and includes some cybersec tools,
   also a community space for users to interact with each other.

 * Tech Stack :
   - Angular
   - Node.js
   - Express
   - MongoDB (mongoose)

 * Features :
   - Access updated CVEs
   - View Latest Articles By Category
   - Interact with community

 * Technical features :
 
 - Backend : Node.js

   * Express Framework 
   * Cors Configuration
   * CRUD RESTFul APIs
   * External APIs (NVD/NewsAPI)
   * Scheduled Cron-Job to fetch and store data
   * JWT Authentication & Authorization
   * Cohere AI Integration to classify data
   * mongoose ODM 
   * MongoDB Atlas Connection
 
 - Frontend : Angular 

  * Tailwind Css
  ...
