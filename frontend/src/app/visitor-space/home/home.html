What is CVE ?

CVE (Common Vulnerabilities and Exposures) is an international standard that identifies and catalogs publicly known cybersecurity vulnerabilities. Each CVE entry provides a unique ID, a description of the vulnerability, and references for further details. CVE helps security professionals, organizations, and researchers share information consistently and respond faster to security threats.
